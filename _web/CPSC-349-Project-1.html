Project Summary

Group Project for Web Front-End Engineering. Uses Eleventy, a static site generator, and template language Liquid to host a website. Pagination is used to display each of the images retrieved from a JSON obejct in its own HTML page.


Responsibilities

  • Used pagination to create individual HTML pages for each of the photos retrieved from a JSON object
  • Used a CSS stylesheet (mvp.css) to style semantic HTML elements
