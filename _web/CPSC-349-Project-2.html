Project Summary

Group Project for Web Front-End Engineering. A dynamic website that uses the open-source framework Remake to create a no-code academic planner (todo-list). This project only uses HTML and CSS. To add items that will be saved and loaded when the user logs in, Remake's attributes are added to an HTML element's class attribute.


Responsibilities

  • Created and stylized the navigation bar
  • Stylized the about and home pages
  • Used Tailwind to style all the lists/subject/items components
  • Identified and solved issues with subjects and items not saving after being created
  • Fixed issue with saved subjects and title not appearing after user logged out
